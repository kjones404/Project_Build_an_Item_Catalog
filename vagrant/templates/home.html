Welcome to Fresh Tomatoes v2.0

Welcome to the new Fresh Tomatoes. The original Fresh Tomatoes was created for the first project in the Udacity Full Stack nanodegree. This rebuild includes some new features that are required for the forth project in the program. These features include:

  • A JSON Endpoint
  • CRUD Functionality:
    • Read
    • Create
    • Update
    • Delete
  • Authentication

This website uses 3rd party applications such as Flask & SQL Alchemy for CRUD functionality, along with Google’s OAuth for authentication. The Movie Database’s API is also utilized to provide fast and accurate movie data when creating or adding to your movie collections.
